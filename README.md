## Running

Clone the project

```bash
git clone https://github.com/lsferreirapel/crud-bash.git
```

Open project folder

```bash
cd crud-bash
```

Install the dependencies

```bash
npm install
```

Run 

```bash
npm run start
# or
node index.js
```
